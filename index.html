How to Play Drawblins

Objective

One player describes a monster while others draw it based on their description!

ğŸ“‹ Setup

  • Use "Player Order" to determine who goes first
  • Adjust difficulty and timing in Settings
  • Give each player paper and drawing materials

Gameplay

  1. Study Phase: Active player memorizes the monster image
  2. Drawing Phase: Active player describes while others draw
  3. Reveal: Compare drawings to the original monster!
  4. Rotate: Next player becomes the describer

Tips for Describers

  • Start with overall shape and size
  • Describe distinctive features clearly
  • Use simple, clear language
  • Guide step-by-step, don't rush
